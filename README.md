# ⟡ CollabSpace — Student Collaboration Platform

A full-stack web application for student collaboration built with Node.js, Express, MySQL and Socket.io.

## 🚀 Features
- 👤 User Authentication (JWT)
- 💻 Project Collaboration
- 📚 Study Groups
- 📄 Resource Sharing
- 💬 Real-time Chat (Socket.io)
- 🔔 Notifications
- 📸 Profile Photo Upload
- 🎓 Academic Tracker
- 🔍 Global Search
- 🌙 Dark/Light Mode

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express.js
- **Database:** MySQL
- **Real-time:** Socket.io
- **Auth:** JWT + bcrypt

## ⚙️ Setup
\`\`\`bash
npm install
cp .env.example .env
mysql -u root -p < schema.sql
node server.js
\`\`\`
